# Roomie - MIT Manipal Roommate Finder

A premium web application for Manipal Institute of Technology students to find roommates and shared accommodation.

## 🚀 Quick Start

### Backend
1. `cd backend`
2. `npm install`
3. Edit `.env` with your MongoDB and Google OAuth credentials.
4. `npm start` (or `npm run dev` if you have nodemon)

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Lucide React
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **Auth**: Google OAuth (Passport.js/Firebase), JWT

## 📁 Project Structure
- `backend/src/`: Controllers, Routes, Models, Middleware, Config, Services
- `frontend/src/`: Components, Pages, Hooks, Context, Utils
